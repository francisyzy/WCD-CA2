Project: Branding for a Local restaurant at ION Orchard

This project is designing artwork for a local fictional high-end restaurant that is serving local dishes. It this project commenced in April of 2015 and ended around early June of 2015. The project is to access the skills and techniques learned by the students in the previous four topics. The software used for this project is Adobe Photoshop CS6 and Illustrator CS6. The project is designed to be printed on A3 papers.
