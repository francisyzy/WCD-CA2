Project: Singapore Science and Engineering Fair (SSEF) 2015

This project is designing artwork for a local fictional event about science and engineering. It this project commenced in March of 2014 and ended around May of 2014. The project is to teach students how to use visual design and software. The software used for this project is Adobe Photoshop CS6 and Illustrator CS6, but it was not utilised in this project.
